## 📝 学习笔记

### 常用数据结构
- **HashMap**: 快速查找，时间复杂度O(1)
- **TreeMap**: 有序存储，时间复杂度O(log n)
- **ArrayList**: 动态数组，支持随机访问

### 算法技巧
1. **双指针**: 处理数组、字符串问题
2. **哈希表**: 快速查找、去重
3. **递归**: 树、图的遍历
4. **动态规划**: 最优化问题
